# User Flows

## Farmer flow

1. Sign in / sign up
2. Open farmer dashboard
3. Check weather, advisory, and crop planning
4. Use AI crop scanner for disease/pest help
5. Ask SmartChatbot for scheme or crop support
6. View mandi/analytics insights
7. Manage calendar, documents, and profile

## Expert flow

1. Sign in as expert
2. Access expert dashboard
3. Review AI advisory context and scanner outputs
4. Use chat and notifications to support users
5. Provide domain guidance and monitor incoming requests

## Merchant flow

1. Sign in as merchant
2. Access merchant dashboard
3. Discover farmers and crop interests
4. Track market prices
5. Manage merchant-side documents and notifications

## Admin flow

1. Sign in as admin
2. Access admin dashboard
3. Review users, analytics, notifications, and settings
4. Verify documents and oversee system-wide activity

## Demo walkthrough order

Best presentation order:

1. Landing page
2. Login / role selection
3. Farmer dashboard
4. AI crop scanner
5. Smart chatbot
6. Calendar and advisory panel
7. Analytics dashboard
8. Field map and NDVI profile
9. Merchant/Expert/Admin snapshots
